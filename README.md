Installing on OS X
==================

1. Install [Homebrew](http://brew.sh/)
1. From a terminal, run

   ```
   brew install python3
   pip3 install -r requirements.txt
   ```

Running on OS X
===============

1. Run

   ```
   python3 main.py "tissue cryopreservation"
   ```

   The output will be in `out.csv`.
